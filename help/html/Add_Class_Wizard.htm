Add Class Wizard

The Add Class Wizard is used to generate class include (inc) and source (clw) files. The files generated are based on templates, simple text files that provide the basic framework for the output. The templates can also define prompts that are used by ClassViewer to create entry controls on the Add Class Wizard dialog screen. The user's input can then be used in the generated source.

 

 

.inc Template

Use the dropdown to select from the list of registered inc templates. This template will be used to provide the basic framework for the generated include (inc) file. Press the edit button to edit the currently selected template. Click here to read more about templates.

 

.clw Template

Use the dropdown to select from the list of registered clw templates. This template will be used to provide the basic framework for the generated source (clw) file. Press the edit button to edit the currently selected template. Click here to read more about templates.

 

Next

Press the next button to continue to the second part of the Add Class Wizard process.

 

Cancel

Press the cancel button to return to the Main window.

 

 

 

Category

Select from the list of know class category (template family) choices or enter a new value of your own. The data entered here provides the value for the %CATEGORY template token.

 

Class Name

Enter the name of the class to be generated. The data entered here provides the value for the %classname template token. If the .inc File or the .clw File fields are empty, the program provides a default name based on the entered Class Name.

 

Base Class

If the new class is to be derived from another class, select the base class from the list of known classes. If a base class is selected, the methods of the base class will appear on the methods tab. If no base class is selected, the methods tab is hidden. The data entered here provides the value for the %BASEclass template token.

 

.inc File

Enter the path and filename to be used for the generated include (inc) file. If no path is provided, the file will be created in the current working directory. The data entered here provides the value for the %INCFILEname template token.

 

.clw File

Enter the path and filename to be used for the generated source (clw) file. If no path is provided, the file will be created in the current working directory. The data entered here provides the value for the %CLWFILEname template token.

 

Finish

Once you have entered the required information, press the Finish Button to have ClassViewer generate the source files. Once the generation is completed, a Smart Scan is initiated to add the class to the database and position the viewer to your new class.

 

Cancel

Press the cancel button to return to the Main window.

 

 

 

Interface List

Select the inferface(s) that will be implemented by the class. .inc File. Code for the selected interfaces is placed where the %INTERFACES template token is found.

 

Generate Interface Callbacks

Check this option to have additional interface callback methods added to the class.

 

 

 

Method List

If a base class is selected, the methods of the base class will appear on the methods tab. If no base class is selected, the methods tab is hidden. Use the method list to select the methods to be derived from the previously selected base class. You can select/deselect items with a single left mouse click. A right mouse popup menu is also provided to select/deselct all methods in the list. Code for the selected methods is placed where the %METHODS template token is found.
