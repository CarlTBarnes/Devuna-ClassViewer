Scan Options

The Scan Options dialog is displayed whenever the user initiates a database refresh.

 

 

Redirection File

The redirection file is the primary mechanism used by Class Viewer to determine what folders need to be scanned. The default value is the standard redirection file for the version of Clarion that has been selected. The user can override this file prior to scanning by pressing the ellipsis button to the right of the field. A file dialog is displayed to allow the user to select a redirection file.

 

Current Directory

Inside the redirection file it is common to use a '.' to represent the current directory. To use a path other that the program startup folder as the current directory, press the ellipsis button to the right of the field. A file dialog is displayed to allow the user to select a folder.

 

Include Standard Equate Files

Use the check boxes it select the standard equate files to be included in the scan. By default, the scan process only scans the class inc and clw files. There are several other standard files that contain items that may be of interest to the user and the check boxes provide a convenient way to specify the inclusions desired.

 

Additional Files

Press the Additional Files button to display the Additional Files dialog which allows the user to include specific files.

 

Scan All Source Files

Check this option for force the program to rescan all source files. This can be a lengthy process depending on the number and size of the files to be scanned. Leave this option unchecked to use the Smart Scan feature and only files that are new or have changed since the last scan will be processed.

 

Scan Now

Press the Scan Now button to initiate the source file scanning process. A progress bar is displayed to inform the user of the scanning progress. When completed, the user is returned to the Main window.

 

Cancel

Press the Cancel Button to discard your changes and return to the Main window.
