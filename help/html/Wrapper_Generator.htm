Wrapper Generator

The Wrapper Generator is used to generate a class wrapper template (tpl) file for the selected Class. The file is generated based on templates, simple text files that provide the basic framework for the output.

   

 

Wrapper Generator Template

Select a registered Wrapper Generator template from the drop down list. You can have as many wrapper generator templates as you like; this allows the programmer to develop different wrapper templates for different purposes. The sample template provided is a general purpose wrapper generator. The template that results from using the example template can be dropped into you application multiple times, either globally of locally.

 

Press the ellipsis to the right of the drop list to Edit the selected Wrapper Generator template file.

 

Generated Template File

Enter the path and filename for the template file to be generated.

Press the ellipsis to the right of the entry field to bring up a file dialog.

 

OK

Once you have entered the required information, press the OK Button to have ClassViewer generate the class wrapper template.

 

Cancel

Press the cancel button to return to the Main window.
