Edit Favorites

The Favorites Menu Dialog allows the user to register other programs with the ClassViewer to allow easy access to these applications from the menu system or the Favorites Toolbar.

 

 

Insert

Press the Insert Button to register a new program for use with ClassViewer. Enter the desired menu text and press the Tab key. A file selection dialog is displayed. Select the desired file and it will be added to the list. You can Edit the Favorites Menu entry by double clicking on it in the list.

 

Delete

Press the Delete Button to remove the Favorites Menu entry from the list.

 

UpRow / DownRow

Press the Up and Down Row buttons to reorder the entries in the Favorites Menu.

 

OK

Press the OK Button to save your changes and return to the Main window

 

Cancel

Press the Cancel Button to discard your changes and return to the Main window.

 

Double-Click

Double-Click on an item in the list to edit the Menu text and Path.
