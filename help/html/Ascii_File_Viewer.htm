Ascii File Viewer

The Ascii File Viewer is the component of the program that provides a means for the user to easily view the selected item in the context of the selected item's source file(s).

 

There are three user definable colors that are used by the Ascii File Viewer: Selected Background, Highlight Color 1, and Highlight Color 2. [See: Edit Options].

 

  • The Selected Background color highlights the currently selected source line.

  • Highlight Color 1 highlights the initial source line selected when the viewer was started.

  • Highlight Color 2 highlights source lines that reference the item that was selected in the Tree View when the viewer was started.

    •  

    The highlighted source lines are shown on a COLOR:BTNFACE background except when the source line is also the currently selected source line; in which case the Selected Background color is used.

     

     

     

    List Box

    Use the scroll bars to view the source file. Double Click to switch to Text Box mode.

     

    Text Box

    The source file is available in a text control to allow the user to select and copy portions of the source file.

     

    Function Buttons

    Top - Move to the first line of the file.

    Page Up - Move 1 page up in the file.

    Page Down - Move 1 page down in the file.

    Bottom - Move to the last line of the file.

    Go To - displays a small dialog box to allow the user to jump to any line number.

    Find - displays a small dialog box to allow the user to search for specific text.

    Print - displays a print options dialog to allow the user to print all or part of the file.

    Edit - launches the user defined Editor.

    Text - switch to the text view of the document.

    List - switch to the list view of the document.

     

    Hot Keys

    Ctrl+G

    display GoTo dialog

    Alt+F3

    display Find dialog

    Ctrl+F

    display Find dialog

    F3

    Find Next match

    Shift+F3

    Find Previous match

    Ctrl+F3

    Find Next reference (when launched from Call Tree view)

    Ctrl+Shift+F3

    Find Previous reference (when launched from Call Tree view)

    Ctrl+P

    display Print dialog

    Ctrl+E

    launch Editor

    Ctrl+T

    switch to Text Mode

    Ctrl+L

    switch to List Mode

     

    Popup Menu

    Viewer functions can also be accessed from a popup menu. Right-click on the viewer display to get the menu to appear.

     

    There are two special functions that are only available on the popup menu:

    Toggle All Folds - collapses or expands all folds

    FoldMargin - hides or shows the fold margin.

     

    The fundamental operation in folding is making lines invisible or visible. Line visibility is a property of the view rather than the document. From the point of view of the user, lines are hidden and displayed using fold points. Generally, the fold points of a document are based on the hierarchical structure of the document contents.

     

    Close

    Close the Ascii File Viewer window.
