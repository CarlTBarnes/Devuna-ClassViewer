Browse Database

The Browse Database window provides structured access to the underlying data used by the Class Viewer program. There are three tabs in the browse window to simplify data display.

 

The Class Tab displays the class name, parent name, and information on the file locations and line numbers. The data can be ordered by either Name or Parent by simply clicking on the corresponding listbox header.

 

 

The Method Tab displays the class name, the sorting name of the method, the method name, prototype and scope information and line numbers. The data can be ordered by either Class, Sort or Name by simply clicking on the corresponding listbox header.

 

 

The Property Tab displays the class name, property name, data type and information on the file locations and line numbers. The data can be ordered by either Class, Name or Data Type by simply clicking on the corresponding listbox header.

 

 

Close

Press the Close Button to return to the Main window.
