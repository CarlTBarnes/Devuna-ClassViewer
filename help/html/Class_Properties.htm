Class Properties

The Edit Class Properties dialog allows the user to modify the Category and Detail Level for a Class. The dialog is displayed when the user selects Properties from the Popup Menu.

 

 

Class Name

The Class Name is displayed for reference purposes and cannot be changed.

 

Category

The field is only available for editing on base classes (ie classes that are not derived). For filtering purposes, all derived classes are considered to be in the same Category as the base class from which they are derived.

 

To specify a category for a non-base class, you must use one of the following options:

 

  • The category is extracted from the optional FamilyClass parameter that can be added to the !ABCIncludeFile(FamilyClass) comment line that identifies ABC Classes.

  • For Non-ABC Classes the program also recognizes a !Category(Category) comment line.

 

Detail Level

Use the spin control or enter a number in the range 0 to 9 where 0 represents the least and 9 represents the most detail.

 

The detail level allows the program to fine tune how much detail is presented in the tree view. Most classes are assigned a detail level of 1 when they are initially scanned. There are some basic classes [see list] that are assigned detail level 0. This was done to allow the program to present a simple view to the beginner while providing a feature for advanced users that permits fine tuning of the amount of detail presented in the tree view. This feature would be particularly useful in controlling the presentation of derived classes.

 

OK

Press the OK Button to save your changes and return to the Main window.

 

Cancel

Press the Cancel Button to discard any changes you have made and return to the Main window.
