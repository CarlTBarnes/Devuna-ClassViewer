Template Files

The Template Files dialog is used to register the templates that may be used by the Add Class Wizard. There are two types of templates inc for include file generation and clw for source file generation. They are maintained in separate registry lists but use the same dialog for maintenance.

 

 

Insert

Press the Insert Button to register a new template for use with ClassViewer. A file selection dialog is displayed. Select the desired file and it will be added to the list. The base filename is used to create the template name. You can change the template name by double clicking on it in the list.

 

Delete

Press the Delete Button to remove the template file from the list. The file is not deleted.

 

Edit

Press the Edit Button to edit the currently highlighted template.

 

OK

Press the OK Button to save your changes and return to the Edit Options window

 

Cancel

Press the Cancel Button to discard your changes and return to the Main window.
