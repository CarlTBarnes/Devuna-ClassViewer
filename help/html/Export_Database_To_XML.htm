Export Database To XML

The Export Database to XML dialog allows the user to select which of the application's internal data queues to export. These are raw data dumps of the queues that are used by the program and stored in the application databases.

 

 

 

Panel Controls

  • Use the check boxes to select the queues to be exported.

  • Use the entry fields to specify the filename to be used for each queue selected.

  • The ellipsis button to the right of each entry field displays a file dialog to allow the user to select an existing file.

 

Export

Press the Export button to start the process. As each queue is processed, an icon appears to the left of the check box to inform the uesr of the export status.

Cancel

Press the Cancel button to close the dialog window and return to the Main window.
