Additional Files

The Additional Files dialog allows the user to specify additional files to be scanned into the database. This is particularly useful for files that are located in folders that are not referenced in the redirection (*.red) file.

 

 

Insert

Press the Insert Button to add a new file to the list of additional files. A file dialog is displayed to allow the user to select a file.

 

Delete

Press the Delete Button to delete the currently highlighted file from the list of additional files. A confirmation dialog is displayed.

 

OK

Press the OK Button to save your changes and return to the Edit Options dialog.

 

Cancel

Press the Cancel Button to discard any changes you have made and return to the Edit Options dialog.

 

Double-Click

Double-Click on an item in the list to edit the file path.
